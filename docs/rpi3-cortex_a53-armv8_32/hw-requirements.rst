Hardware Requirements
=====================


* **A Raspberry PI v3 board** Ensure that the model B or B+ with the Cortex-A53 quad-core processor

* **Hardware Virtualization extensions**
  
  * **Cortex A53**\ : Hardware Support for Virtualization

* **2nd-level page tables** Typically turned on implicitly along with Virtualization extensions, if the processor supports it.
  
  * **Cortex A53**\ : Second-stage Page Tables
